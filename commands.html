Commands List

North, East, South, West, Up and Down:

'North/East/South/West/Up/Down'
  • These commands move the player to the location in the direction specified.
  • Available directions are listed in each location as places you can go.

Hint:

'Hint'
  • This command gives you a hint for when you are stuck.

Break:

'Break + item '
  • This command allows you to break items.
  • The item must either be in your inventory or current place
  • E.g Break chair

Jump:

'Jump + direction'
  • This command allows you to jump around the map in any direction.
  • Use this to move between places when a jump is required
  • E.g Jump West

Attack:

'Attack + item + item(weapon)'
'Attack + door/exit + direction + item(weapon)'
  • This command allows you to attack items with items in your inventory.
  • E.g Attack bookshelf axe

Drink:

'Drink + item(drinkable)'
  • The item must have the drinkable property
  • The item must be in your inventory or in the current place
  • Drinking something increases or decreases your stamina by 5 depending on if it's poisonous or not.

Eat:

'Eat + item(edible)'
  • The item must have the edible property
  • The item must be in your inventory or in the current place
  • This command increases or decreases your health by 5 depending on if it's poisonous or not.

Examine:

'Examine + item'
  • This command will provide you with a description of an item
  • It will also give you an idea towards it's properties

Open:

'Open + item(container)'
  • This command will open an item with the container property and reveal any items inside.
  • The container must be unlocked
  • E.g Open wardrobe

Take and Drop:

'Take + item'
'Drop + item'
  • This command allows you to add or remove items from your inventory.
  • You can't pick up items if you are carrying too much or if they are too heavy.

Unlock:

'Unlock + item'
'Unlock + door/exit + direction'
  • Unlocks an item or exit.
  • Unlocking an item allows it to be opened to view items inside
  • Unlocking an exit allows you to pass through it in that direction

Climb:

'Climb + direction'
  • This command allows you to climb in different directions.

Dig:

'Dig'
  • This command allows you to dig into the ground to reveal hidden items.
  • Requires a shovel in the inventory

Push:

'Push + item'
  • This command will allow you to push an item to reveal items that are hidden.

Fix:

'Fix + item'
  • This command allows you to fix a broken item.

Burn:

'Burn + item'
  • This command allows you to set an item on fire.
  • It will lose durability over time until it breaks.
